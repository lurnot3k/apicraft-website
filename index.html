A propos du Bot

Ce bot est axé sur le côté données de Minecraft.
Il va vous permettre de connaître les infos sur un profil Minecraft
tout comme savoir les infos instantanées de certains serveurs.

COMMANDES


-profil [pseudo] pour afficher les profils des joueurs.
-paladium Pour afficher les infos sur le plus grand serveur pvp/faction français, contient :
Le nombre de joueurs connectés, la version du serveur et le statut
-hypixel Pour afficher les infos sur le serveur le plus important de Minecraft, contient :
Le nombre de joueurs connectés, la version du serveur et le statut.
